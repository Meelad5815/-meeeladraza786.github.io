WordPress Portfolio Guide

The Master Portfolio Prompt

Create a high-end, fully functional professional portfolio website for your role (e.g., Web Developer/Designer) with these sections:

  • Hero Section: Bold headline, professional bio, and a clear CTA leading to the contact page.
  • Portfolio Gallery: Filterable project grid with hover effects and links to case studies or live demos.
  • Services/Skills: Interactive expertise area using icons and progress bars.
  • About Me & Resume: Experience overview with a "Download CV" button.
  • Testimonials: Slider showcasing client feedback and social proof.
  • Contact Form: Name, Email, Subject, Message fields with email notifications.
  • Blog/Insights: Dynamic layout for recent articles or updates.
  • Footer: Social links, copyright, and quick navigation.

Technical Requirements: SEO optimized, fast loading, mobile-first, and smooth scrolling animations.

Tips for Your WordPress Setup

  1. Choose a Theme: Go to Appearance > Themes and search for "Astra" or "OceanWP".
  2. Use a Page Builder: Install Elementor for drag-and-drop sections.
  3. Import a Starter Template: Use a portfolio starter site to get a fast setup.
